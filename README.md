# SusWord

Real-time multiplayer social deduction word game.

## Tech Stack (Planned)
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express + Socket.IO

## Game Overview
SusWord is a social deduction game where 3-8 players are given a secret word. One player is the "Imposter" and receives a similar but different word. Players take turns giving clues to identify the imposter. If the imposter is caught, they have one final chance to guess the main word to win.
